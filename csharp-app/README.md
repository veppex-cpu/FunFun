# C# Console App

This project targets `net8.0`, which is newer than .NET 7.0.

## Prerequisite

Install the .NET SDK 8.0 or newer.

On macOS with Homebrew:

```sh
brew install --cask dotnet-sdk
```

Or download it from Microsoft:

https://dotnet.microsoft.com/download

## Run

From this directory:

```sh
dotnet restore
dotnet run
```

From the workspace root:

```sh
dotnet run --project csharp-app
```
